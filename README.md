### 🔧 [Self-Healing Infrastructure Monitoring System](https://github.com/sachinxsharma/self-healing-monitor)

</div>

> A system that doesn't just monitor your infrastructure — **it fixes itself.**

```
Container crashes?     →  Auto-restarted in < 30 seconds
Disk above 85%?        →  Old logs auto-cleaned
CPU / RAM spike?       →  Email alert fired instantly
Code pushed?           →  Auto-built and deployed via CI/CD
```
